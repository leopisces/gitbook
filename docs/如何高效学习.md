# 如何高效学习
> 如果只用某一种方式了解某样事物，你就不会真正了解它。了解事务真正含义的秘密取决于如何将其与我们所了解的事物相联系。通过联系，你可以将想法内化于心，从各种角度看问题，直至找到适合自己的方法。这才是思考的真谛！  

<span style="float:right;">--马文.明斯基</span> 

## 整体性学习策略

### 结构
- 结构就是一系列紧密联系的知识
- 理解就是结构高度发达完善的结果
- 建立知识结构需要尽可能多的建立尽可能多的联系

### 模型
- 是简化的结构
- 是对信息的压缩
- 是知识最核心的概念
- 类似于思维导图

### 高速公路
- 高速公路作为参照物可以连接两个完全不同的观念
- 能激发创造力
